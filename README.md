# How to open project
`git clone https://github.com/weedprotector/Employees_table.git`
## Run backend:

### `cd Employees_table_backend`
Rename `.env.example` to `.env`
### Install dependencies: `npm i`

Run: `npm run start:dev`


## Run frontend:
In another terminal
### `cd employees_table_frontend`
Rename `.env.example` to `.env`
### Install dependencies: `npm i`

Run: `npm run start`
